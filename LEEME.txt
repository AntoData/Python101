Requisitos para ejecutar el proyecto:

- Python 3.6 o 3.7:
	
	En caso de no tener python instalado o en tales versiones, por favor vaya a 
	
	www.python.org y descargue la versión adecuada a su sistema operativo


- pip (en realidad es opcional si ya tiene instaladas el resto de librerías de python en su sistema)

	Para obtenerlo, descárguese el fichero get-pip.py (de aquí lo puede obtener: https://bootstrap.pypa.io/get-pip.py)
 	abra un terminal (o cmd en Windows) vaya a la ruta donde haya guardado el fichero get-pip.py y escriba:
	python get-pip.py
	
Tenga en cuenta que "python" es el alias por el que ejecuta el ejecutable de python
	
	Puede ser también python3 o python3.6 o python3.7


- pygame 1.9.6 (librería de python no estandar):

	Para obtenerla y una vez que haya instalado pip de manera muy sencilla. Abra
	
	un terminal (o cmd en Windows) y escriba:

	pip install pygame

	Pulse intro para ejecutar el comando.

	Tenga en cuenta que puede necesitar permisos de administrador, por lo que o bien

	ejecute el terminal o cmd como administrador o en ubuntu diga: sudo pip install pygame
	
	Como esta es la última versión de pygame, no debería tener problemas ya que será la que
	se instale.
	
	En caso de tener una versión previa instalada, desinstálela y ejecute el comando


- pandas 0.24.2

	Igualmente, una vez que tenga pip instalado, abra un terminal o cmd y escriba:

	pip install pandas

	Pulse intro para ejecutar el comando
	Tenga en cuenta que puede necesitar permisos de administrador, por lo que o bien

	ejecute el terminal o cmd como administrador o en ubuntu diga: sudo pip install pandas

	Como esta es la última versión de la librería, no debería tener problemas ya que será la

	que se instale.
	En caso de tener una versión previa instalada, desinstálela y ejecute el comando


- matplotlib 3.1.0

	De nuevo, una vez que tenga pip instalado, abra un terminal o cmd y escriba:
	
	pip install matplotlib
	
	Pulse intro para ejecutar el comando
	
	Tenga en cuenta que puede necesitar permisos de administrador, por lo que o bien

	ejecute el terminal o cmd como administrador o en ubuntu diga: sudo pip install matplotlib

	Como esta es la última versión de la librería, no debería tener problemas ya que será la

	que se instale.
	En caso de tener una versión previa instalada, desinstálela y ejecute el comando

- Fuentes Arial y Futura PT (opcional):
	Para una correcta visualización ambas fuentes deberían instaladas como fuentes del
	sistema operativo. Por defecto, los sistemas operativos mencionados llevan tales
	fuentes. En caso de no estar, el programa se ejecutará igualmente pero se visualizará
	la fuente por defecto.

Ejecución:
Una vez haya satisfecho tales requisitos podrá ejecutar el programa sin ningún problema.

Para ello ejecute el archivo app.py de la carpeta principal

Consejos:

En Windows solo tendrá de hacer doble click sobre tal fichero

En Ubuntu lo más frecuente es que tenga que abrir un terminal, ir hasta la ruta y ejecutar:
python app.py
(Teniendo en cuenta que python es su alias para el ejecutable de python que podría ser también
python3 o python3.6 o python3.7)
